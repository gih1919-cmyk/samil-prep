# DART 재무비율 자동계산 & 웹 대시보드

DART Open API를 활용해 상장사·비상장사의 재무제표를 자동으로 조회하고, 7가지 재무비율을 계산하여 웹 대시보드로 보여주는 프로그램입니다. 엑셀 다운로드 기능도 함께 제공합니다.

## 주요 기능
- DART Open API 연동을 통한 실시간 재무제표 조회
- 부채비율, 유동비율, 재고회전율, 영업이익률, 순이익률, ROA, ROE 자동 계산
- Flask 기반 웹 대시보드로 결과 시각화
- 엑셀(.xlsx) 파일 다운로드 기능

## 사용 기술
- Python (requests, pandas, openpyxl)
- Flask (웹 서버)
- DART Open API

## 프로젝트 구조
- `Company` 클래스: 기업 단위로 재무데이터 조회·계산 로직을 캡슐화
- `get_account()`: 계정과목명이 기업/연도별로 다르게 표기되는 문제에 대응하기 위한 범용 조회 함수
- Flask 라우트: `/`(대시보드 화면), `/download`(엑셀 다운로드)

## 개발 과정에서 해결한 주요 이슈
1. **요약 API의 한계** → 재고자산 등 세부 계정과목이 없어 전체 재무제표 API(`fnlttSinglAcntAll`)로 전환
2. **계정과목명 중복 문제** → 자본변동표 등에서 같은 이름이 반복되어 값이 잘못 덮어써지는 버그 발견, `sj_div`(재무제표 종류) 필터링으로 해결
3. **기업별 표기 차이 대응** → 손익계산서 제출 형식(IS vs CIS), 계정과목명 표기 차이(예: "당기순이익" vs "당기순이익(손실)")를 여러 후보를 시도하는 `get_account()` 함수로 방어적으로 처리
4. **API 데이터 범위의 한계 확인** → 정기보고서 미제출 대상 비상장사(예: 우아한형제들)는 조회되지 않음을 실증적으로 확인
5. **보안** → API 키를 코드에서 분리해 환경변수로 관리

## 알려진 한계
- 정기보고서(사업보고서) 제출 의무가 없는 비상장사는 조회되지 않음 (DART API 데이터 범위 한계)
- 해외 상장 기업(예: 쿠팡의 미국 본사)은 국내 DART 시스템에 잡히지 않음
